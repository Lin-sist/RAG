## 诊断与下一闸门

首轮 NVIDIA backend 以 `https://integrate.api.nvidia.com` 作为 rerank base URL。NVIDIA 当前官方 API reference 给出的模型 endpoint 是 `https://ai.api.nvidia.com/v1/retrieval/nvidia/llama-nemotron-rerank-1b-v2/reranking`；其 Retriever quickstart 也区分 embedding 的 `integrate.api.nvidia.com` 与 reranker 的 `ai.api.nvidia.com`。因此“host 漂移/误配”是高置信根因推断，但尚未由真实调用证明。

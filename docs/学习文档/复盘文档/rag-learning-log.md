要求按以下结构输出：
1. 这个模块的职责
2. 它在整个项目调用链中的位置
3. 关键类/方法的作用
4. 为什么要这么设计
5. 面试官可能会怎么追问
6. 最后给我 5 个检查问题
不要一次性讲太泛，优先结合当前代码。

4. 哪些关键点我漏掉了
然后要求我重新组织答案。
